JavaScript Training Tasks:-

This repository contains JavaScript practice tasks created for learning and improving core JS fundamentals.

=> Task 1 — Student Learning Simulator
The first task is a simple student simulation program that demonstrates essential JavaScript concepts.
Included Concepts
- var, let, const
- Basic data types & typeof
- String methods: trim, slice, replace, toUpperCase, etc.
- Operators & type coercion
- Math and Number utilities
- User input with prompt()
- Creating and using objects
- Arrays: add, remove, sort, slice, find
- Loops: for, while, for-of, nested loops
- Generating random grades
- Formatted student report using template literals
- Summary table using console.table

=> Task2 — Smart Library Management System
Main Features Implemented
* Add multiple books to a global library
* Search books by title or author
* Filter by category
* Count how many books each author has
* Show all book titles
* Display complete library statistics
* Create manager object using prototype inheritance
* Merge default and user settings

Functions & Objects
- Creating book objects using functions
- Default parameters
- Nested objects
- Constructor functions (Book)
- Object.create() & prototype inheritance
- Object.assign() for merging settings

Arrays & Higher-Order Functions
- map() – get all titles
- filter() – filter by category / keyword
- find() – find a book by ID
- reduce() – count books by author
- forEach() – print all books

Arrow Functions
- Searching books
- Getting old books
- Getting book by ID

Other Concepts
- Lexical scope
- Random element picker
- ID generator using closures
- Dot vs bracket notation
